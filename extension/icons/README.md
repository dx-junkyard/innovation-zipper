# Team Brain Extension Icons

このディレクトリには Chrome 拡張機能用のアイコンを配置してください。

## 必要なファイル

- `icon16.png` - 16x16 ピクセル（ツールバー用）
- `icon48.png` - 48x48 ピクセル（拡張機能管理画面用）
- `icon128.png` - 128x128 ピクセル（Chrome ウェブストア用）

## 注意事項

- アイコンがない場合、ビルドスクリプトが自動的にプレースホルダーを生成します
- 本番環境では適切なデザインのアイコンに差し替えてください
- PNG 形式を使用してください
